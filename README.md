# 💳 Credit Risk Prediction System

A machine learning-based web application developed using **Python**, **Streamlit**, **XGBoost**, and **Scikit-learn** to predict the credit risk of loan applicants. The application analyzes customer financial information, such as income, loan amount, credit history, and other key factors, to estimate the probability of loan default and support data-driven lending decisions.

## 📌 Project Type

**Machine Learning | Predictive Analytics | Finance | Classification**

## 🚀 Features

- Predicts loan default risk using an XGBoost machine learning model.
- Interactive Streamlit interface for real-time credit risk assessment.
- Provides fast and accurate predictions to support financial decision-making.

## 🛠️ Technologies Used

- Streamlit
- XGBoost
- Scikit-learn
